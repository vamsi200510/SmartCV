# Components Directory

This directory is structured to hold reusable React components. To maintain scalability, group components by their functional domain or feature module:

- `/common` - Universal UI elements (Buttons, Inputs, Modals, Cards, Loaders)
- `/auth` - Authentication related widgets (Sign-in form, Sign-up form)
- `/resumes` - Resume building, viewing, and layout tools
- `/ats` - ATS scoring dashboards, match percentage gauges, recommendation cards
- `/ai` - AI prompts, chatbot triggers, suggested resume improvements
