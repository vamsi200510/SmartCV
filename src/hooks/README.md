# Custom Hooks Directory

This directory holds reusable React custom hooks to decouple business logic from the UI components.

## Scalable Structure

- `useAuth.ts` - Manage authentication session, user context, sign-in/out logic.
- `useResume.ts` - Fetch, create, update, and delete resume data.
- `useATS.ts` - Trigger ATS scoring analysis, check validation status.
- `useAI.ts` - Communicate with Gemini API, stream text recommendations.
